# مساحة Cowork

هذا المجلد يحوّل المحادثات المتفرقة مع الذكاء الاصطناعي إلى سير عمل ثابت ومنظم.

## اختر الوضع الصحيح

- `Chat`: للأسئلة السريعة والأفكار الخفيفة والردود القصيرة.
- `Cowork`: الوضع الافتراضي للعمل الجاد الذي يحتاج ملفات وسياق وخطوات متعددة.
- `Projects`: للمشاريع المشتركة بين أكثر من شخص أو لأي سياق يتكرر باستمرار.
- `Code`: للتطوير، التصحيح، المراجعات، والتنفيذ البرمجي.

## القاعدة العملية

استخدم `Chat` فقط عندما يكون المطلوب قصيرًا ومؤقتًا.

استخدم `Cowork` عندما تلمس المهمة ملفات، أو تحتاج سياقًا سابقًا، أو غالبًا ستأخذ أكثر من جولة.

استخدم `Projects` عندما تريد أن يبدأ كل شخص في الفريق من نفس السياق والقوالب.

استخدم `Code` عندما تكون المهمة تقنية أو برمجية بالدرجة الأولى.

## هيكل المجلدات

- `About Me/`: سياقك الشخصي وطريقة عملك.
- `Projects/`: مساحات عمل للمشاريع أو الفرق أو العملاء.
- `Templates/`: القوالب الثابتة مثل أسلوب الكتابة وبداية كل مهمة.
- `Outputs/`: المخرجات النهائية والنسخ المعتمدة وملاحظات الجلسات.

## الإعداد الأولي

1. راجع `About Me/about-me.md`.
2. عدّل `Templates/anti-ai-style.md` حتى يعكس أسلوبك الحقيقي.
3. استخدم `Templates/master-prompt-template.md` كنقطة بداية ثابتة لكل مهمة.
4. انسخ `Projects/Shared Project/` أو استخدم `Projects/WAFI ERP/` كمشروع جاهز.

## بداية سريعة

الصق هذا النص قبل أي مهمة جديدة:

```text

- Cowork/Templates/anti-ai-style.md

```
