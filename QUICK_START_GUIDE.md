## 2️⃣ كيفية تسجيل سند قبض من عميل

### الخطوات:

```
1. افتح القطاع: الخزينة → القبضات
2. انقر على "جديد"
3. أدخل البيانات:
   ├─ العميل: اختر من قائمة
   ├─ المبلغ: 5000 ريال
   ├─ رقم الفاتورة: INV-001
   ├─ طريقة الدفع: نقد/بنك
   └─ الملاحظات: اختياري

4. النظام تلقائياً:
   ├─ ✓ يحدد حساب العميل (114001)
   ├─ ✓ ينشئ قيد يومي متوازن
   ├─ ✓ يسجل المرجع والمرجع الضريبي
   └─ ✓ يحدث الأرصدة

5. انقر "حفظ"
   → النظام ينقل الحالة تلقائياً إلى PENDING
   → بعد الموافقة → POSTED
```

**ماذا يحدث بالخلفية:**
```sql
INSERT INTO journal_entries 
  (id, type, partner_id, linked_account_id)
VALUES 
  ('REC-001', 'RECEIPT', 'CUST-123', '114001');

INSERT INTO journal_entry_lines 
  (account_id, debit, sub_account_id, invoice_ref, customer_id)
VALUES 
  ('112', 5000, 'CUST-123', 'INV-001', 'CUST-123'),
  ('114001', -5000, 'CUST-123', 'INV-001', 'CUST-123');
```

## 6️⃣ أسئلة وأجوبة سريعة

### س: ماذا لو أدخلت مبلغاً خاطئاً؟
ج: يمكنك تعديل السند قبل الموافقة. بعد الموافقة، أنشئ قيد عكسي.

### س: الحساب الذي أريده لم يظهر؟
ج: يجب أولاً تفعيل الحساب من شاشة شجرة الحسابات.

### س: ماذا يعني "متوازن"؟
ج: مجموع المدين = مجموع الدائن (الأصل المحاسبي)

### س: هل يمكن تصحيح قيد منشور؟
ج: نعم، من خلال إنشاء قيد عكسي مطابق.

### س: كيف أتابع معاملة معينة؟
ج: ابحث عن رقم الفاتورة أو رقم الشيك في القيود.

## 7️⃣ الحقول الإجبارية

### لأي حساب مرتبط (CUSTOMER/SUPPLIER/EMPLOYEE/BANK):
- ✓ معرّف الكيان (customer_id/supplier_id/employee_id)
- ✓ رقم المرجع (invoice_no/po_no/payroll_no)
- ✓ المبلغ والنوع (debit/credit)

### لأي حساب عام (GENERAL):
- ✓ المبلغ والنوع فقط
- ✓ يمكن إضافة وصف إضافي

---

## 8️⃣ الأخطاء الشائعة وحلولها

| المشكلة | السبب | الحل |
|--------|-------|------|
| عدم توازن القيد | مجموع المدين ≠ الدائن | تحقق من الأرقام |
| لا يظهر الحساب | الحساب معطّل | فعّل الحساب |
| رقم مرجع مكرر | استخدام نفس الرقم مرتين | غيّر الرقم |
| لا يظهر العميل | العميل غير مسجل | أنشئ العميل أولاً |

---

## 9️⃣ اختصارات مفيدة

| الاختصار | المعنى |
|---------|--------|
| INV | رقم الفاتورة (Invoice) |
| PO | رقم الطلب (Purchase Order) |
| CHQ | رقم الشيك (Cheque) |
| EMP | رقم الموظف (Employee) |
| REF | المرجع (Reference) |
| AR | حساب العميل (Account Receivable) |
| AP | حساب المورد (Account Payable) |
