## 📊 الإحصائيات الرئيسية

```
126 حساب موحد
├─ 5 أنواع محاسبية (ASSET, LIABILITY, EQUITY, REVENUE, EXPENSE)
├─ 2 فئة ميزانية (BALANCE_SHEET, INCOME_STATEMENT)
├─ 85 حساب مع تصنيفات فرعية (67.5%)
└─ 30 حساب مع مراجع ذكية (23.8%)

4 أنواع سندات مدعومة
├─ Receipt (قبض)
├─ Payment (صرف)
├─ Journal (قيود يومية)
└─ Payroll (كشف رواتب)

5 أعمدة ربط في السندات
├─ sub_account_id (حساب فرعي)
├─ invoice_ref (رقم المرجع)
├─ tax_ref (مرجع ضريبي)
├─ customer_id (معرّف الكيان)
└─ bank_account_id (حساب بنكي)
```

## 💡 أمثلة الاستخدام

### سند قبض من عميل
```javascript
{
  type: "RECEIPT",
  partner_id: "CUST-001",
  amount: 5000,
  account_id: "114001",      // ذمم مدينة
  invoice_ref: "INV-2024-001",
  method: "BANK"
}
// النظام ينشئ قيد يومي موازن تلقائياً
```
