```

```

```
كل مرجع يجب أن يكون فريداً:
INV-001, PO-001, CHQ-001, etc.
```

### 3. الربط الصحيح
```
CUSTOMER → 114xxx account → customer_id
SUPPLIER → 211xxx account → supplier_id
EMPLOYEE → 52xxxx account → employee_id
BANK → 112xxx account → bank_account_id
```
